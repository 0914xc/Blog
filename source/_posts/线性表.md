### 线性表的基本操作
```c
// 初始化表
InitList(&L);

// 求表长
Length(L);

// 按值查找操作
LocateElem(L, e);

// 按位查找操作
GetElem(L, i);

// 插入操作
ListInsert(&L, i, e);

// 删除操作
ListDelete(&L, i, &e);

// 输出操作
PrintList(L);

// 判空操作
Empty(L);

// 销毁操作
DestroyList(&L);
```
